# Credit Card Fraud Detection (Machine Learning)

Welcome to my Machine Learning project! In this repository, I've built a predictive model to detect fraudulent credit card transactions using a dataset of real-world data from Kaggle.

## Project Objective
The goal of this project is to differentiate between fraudulent transactions. Since fraud makes up only *0.173%* of the data, this project focused handling unbalanced datasets and optimizing model performance for rare events.

##  Basic Methodologies
* *Exploratory Data Analysis (EDA):* I visualized class distributions and feature correlations using Matplotlib and Seaborn.

* *Data Scaling:* I applied scaling to transaction volumes ensure consistency for ML algorithms.

* *Unbalanced Data Management:* To ensure accurate identification of fraudulent activity, I focused on more metrics than simple accuracy (such as Sensitivity, Recall, and F1 Score). * *Machine Learning Pipeline:* I built an end-to-end Python pipeline from raw data to model evaluation.

##  Tools and Technology Stack
* *Python* (Pandas, NumPy)
* *Scikit-Learn* (Classification algorithms)
* *Visualization:* Matplotlib and Seaborn

##  How to View the Project
You can view the full Python code, data visualizations, and the 0.173% fraud rate analysis by clicking on the Credit_Card_Fraud_Detection.ipynb file in this repository.
